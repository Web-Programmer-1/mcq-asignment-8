# MCQ analysis
This repository contains tools and scripts for analyzing multiple-choice question (MCQ) data. The analysis focuses on various aspects such as question difficulty, discrimination index, and overall test reliability.
## Features
- Calculate item difficulty and discrimination indices.
  - Item Difficulty: Proportion of students who answered the question correctly.
  - Discrimination Index: Measure of how well a question differentiates between high and low performers.
- Generate summary statistics and visualizations.
- Support for various data formats (CSV, Excel).
- Customizable analysis parameters.
- Export results in multiple formats (CSV, PDF).

## Installation
To install the MCQ analysis tool, follow these steps:
1. Clone the repository:
2. ```
   git clone https://github.com/yourusername/mcq-analysis.git
   ```
3. Navigate to the project directory:
4. ```
   cd mcq-analysis
   ```
5. Install the required dependencies:
6. ```
   pip install -r requirements.txt
   ```
## Usage
To run the MCQ analysis tool, use the following command:
```python mcq_analysis.py --input data.csv --output results/
```
