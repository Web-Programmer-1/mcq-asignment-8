Project Overview
-----------------------
Project Name: MCQ Analysis BD
Live URL: https://student-dashboard-alpha-two.vercel.app

Features
------------------------
Learning & Exam System
MCQ Examination System with 4 options per question

Written Examination platform
Matheditor tool for mathematical expressions
Video guidelines and tutorials
Digital books and study resources
Text-based guidelines and notes

-----------------
Frontend:
Next.js with TypeScript
Redux for state management
Server-Side Rendering (SSR)
Tailwind CSS
Matheditor tool

Backend:
--------------
Node.js with Express
TypeScript
MongoDB (NoSQL database)
Husky for git hooks
AWS S3 for image/file storage
